# 卧室风水分析指南

## 这份文件做什么

用于帮助技能在分析卧室时，不只是笼统讲吉凶，而是围绕休息感、压迫感和恢复状态做更具体的判断。

## 分析重点

卧室分析重点看：
- 床位关系
- 压迫感
- 光线与休息感
- 是否安静
- 是否影响睡眠与恢复
- 与门、窗、通道的关系是否顺

## 常见观察点

### 1. 床位
看床头、床尾、床边与门窗、过道、重物的关系是否舒服。

### 2. 压迫感
看上方、侧边、正前方是否容易带来压迫、逼仄或不安定感。

### 3. 光线与安静度
看卧室是太亮、太吵，还是能形成比较稳定的休息环境。

### 4. 动线干扰
看卧室是否经常被穿过、被打扰，或者过于靠近高频活动区。

## 输出建议

可以按这个顺序讲：
1. 已知条件
2. 主要观察
3. 有利点
4. 不利点
5. 可调整点
6. 哪些问题更值得优先处理

## 表达原则

- 多讲休息感和恢复感
- 少讲悬浮玄词
- 多给能落地的小调整建议
