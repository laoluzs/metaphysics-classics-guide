# 分析示例库

这个文件提供示例，不是固定答案模板。作用是让 skill 在实际输出时更像一个成熟项目，而不是只会报框架标题。

---

## 示例 1：八字分析（事业方向）

### 用户输入
```text
出生日期：1995-09-12
出生时间：09:30
性别：男
出生地：江苏南京
想重点看：事业发展
```

### 示例输出骨架
```text
一、基本信息
- 你提供的信息已经足够做一版四柱八字分析草稿。
- 这次采用的是传统子平命理 / 四柱八字分析框架。

二、分析依据
- 按《渊海子平》的常见路径，先看月令，再看日主与整体配置。
- 结合《子平真诠》的思路，这里除了看旺衰，还要看格局是否成、用神是否立得住。

三、核心判断
- 从盘面结构看，事业方向更适合走稳定积累型，而不是高波动、高投机型路径。
- 早期可能有反复试错，但中后期更看重长期能力沉淀。

四、展开分析
- 事业：更适合持续积累、靠专业能力上升的路径。
- 财运：偏向通过正向积累获得，而不是短期暴起。
- 变化点：真正的转折点往往不在一开始，而在经历一次方向调整之后。

五、边界说明
- 以上为传统术数视角下的分析草稿，仅供参考。
```

## 示例 2：八字分析（感情方向）

### 用户输入
```text
出生日期：1998-03-21
出生时间：19:00
性别：女
出生地：湖南长沙
想重点看：感情和婚姻
```

### 示例输出骨架
```text
一、基本盘面
- 这次重点看感情与婚姻倾向，不做无限外推。

二、分析依据
- 按《滴天髓》强调的旺衰和气势思路，感情判断不只看单一点，还要看整体结构是否顺。
- 《三命通会》的类例里，类似结构往往更重“阶段变化”，不宜一句话论定。

三、核心判断
- 感情上更容易出现“前期犹豫、后期明确”的倾向。
- 真正影响关系稳定的，不只是桃花多少，而是结构里能否形成稳定配合。

四、建议
- 更适合看长期相处后的稳定度，而不是一时热度。

五、边界说明
- 以上为传统命理视角下的分析，仅供参考。
```

## 示例 3：占卜解读（合作能不能成）

### 用户输入
```text
问题：这个合作项目还能不能推进？
时间：2026-04-07 17:00
背景：已经谈了两轮，对方最近回复变慢
想重点看：还能不能成，卡点在哪
```

### 示例输出骨架
```text
一、问题确认
- 你这次问的是合作项目后续能不能继续推进，以及卡点在哪里。

二、占断依据
- 若按《周易》的变化观，这件事更像是进入了“未定而变”的阶段，不是立刻定死的结构。
- 从《梅花易数》的取象思路看，对方回复变慢，本身就是一个节奏变化信号。
- 若借《增删卜易》的案例化思路，重点不只是能不能成，还要看当前阻力是不是可解。

三、结论
- 结果倾向：并非彻底无望，但推进阻力确实在增加。
- 当前状态：更像卡在节奏和信任层，而不是完全谈崩。
- 变化点：后续是否能成，关键在于能否出现一次重新确认条件的机会。

四、建议
- 与其继续空等，不如主动把关键条件再确认一遍。

五、边界说明
- 以上为传统术数解读，仅供参考。
```

## 示例 4：走势预测（未来半年）

### 用户输入
```text
分析对象：我的工作发展
当前状态：最近在考虑换岗
时间范围：未来半年
想重点看：事业走势和变化点
```

### 示例输出骨架
```text
一、对象与范围
- 这次重点看的是未来半年的事业走势。

二、阶段走势
- 当前阶段：更多是犹豫和评估，不宜急着定终局。
- 过渡阶段：会出现一次方向更清晰的窗口。
- 后续阶段：如果前面做出正确选择，后面会进入相对稳定的推进期。

三、分析依据
- 按《滴天髓》重气势流通的思路，这类走势判断关键不在一时起落，而在整体气势有没有转顺。
- 从《三命通会》的类例角度看，这种结构往往先有波动，再有定向。

四、总结
- 整体走势偏“先抑后稳”。
- 真正要注意的是过渡阶段的选择。

五、边界说明
- 以上为传统术数视角下的分析，仅供参考。
```

## 示例 5：风水分析（住宅格局）

### 用户输入
```text
分析对象：住宅
朝向：坐北朝南
当前情况：入户见客厅，厨房在西北角，卧室在东南角
想重点看：整体格局和财位
```

### 示例输出骨架
```text
一、对象确认
- 这次看的是住宅格局，重点看整体关系和财位倾向。

二、分析依据
- 按《葬书》强调的“藏风聚气”思路，先看整体是否聚而不散。
- 从《地理人子须知》的形势观察角度看，重点不只是单点方位，而是入口、动线和功能区关系。

三、主要观察
- 有利点：整体朝向清晰，主要功能区分布相对明确。
- 需要注意：厨房在西北角，可能会成为这个格局里需要重点观察的点。
- 动线：如果入户后空间过于直冲，容易有“聚不住”的问题。

四、调整建议
- 先看入口到客厅的过渡是否太直、太空。
- 如果要看财位，最好结合更具体的平面图进一步判断。

五、边界说明
- 以上为传统风水视角下的分析草稿，仅供参考。
```
