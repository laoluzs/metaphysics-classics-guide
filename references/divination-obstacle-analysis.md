# 占卜阻力点分析指南

## 这份文件做什么

用于帮助技能在分析占卜问题时，不只是说“有阻力”，而是把阻力的来源、性质和可解程度讲清楚。

## 分析重点

很多占卜问题的关键，不只是“能不能成”，而是：
- 卡在哪里
- 谁在拖慢节奏
- 问题是出在条件、时间、态度，还是外部环境
- 当前阻力是短期波动，还是更深层的结构问题

## 常见阻力类型

- 节奏不一致
- 条件没谈拢
- 对方态度反复
- 外部环境变化
- 自身判断还不够清楚
- 第三方干扰

## 常见观察点

### 1. 阻力来源
先看问题主要来自自己、对方、第三方，还是外部现实条件。

### 2. 阻力性质
看它是短期拖延、误会、反复，还是更根本的不匹配。

### 3. 可解空间
看问题是还能谈、还能等、还能调整，还是本身已经很难扭转。

### 4. 后续变化点
看阻力会不会在后续某个节点松动，还是继续拖延。

## 输出建议

可以按这个顺序讲：
1. 当前阻力在哪里
2. 是短期阻力还是结构性阻力
3. 有没有可解空间
4. 后续最关键的变化点
5. 更适合的应对方式
