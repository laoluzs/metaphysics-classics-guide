# 八字感情分析指南

## 分析重点

感情分析不要只盯“有没有桃花”，更应该看：
- 关系模式
- 情感表达方式
- 稳定度
- 阶段变化
- 容易出现的问题类型

## 常见观察点

### 1. 结构整体是否顺
有些盘不是没有感情机会，而是结构本身更容易反复。

### 2. 官杀 / 财星 / 食伤等关系
不同体系会有不同侧重点，解释时要尽量温和，不要绝对化。

### 3. 日主与整体配置
看是更主动、被动、谨慎，还是容易情绪化。

### 4. 大运流年变化
很多关系变化是阶段引动，不是单点决定。

## 输出建议

可以按这个顺序讲：
1. 感情总体倾向
2. 相处模式特点
3. 容易出现的矛盾点
4. 更适合的关系节奏
5. 阶段变化提醒

## 表达原则

- 少下重话
- 少做命定式判断
- 多讲关系模式与阶段变化
- 避免制造焦虑
