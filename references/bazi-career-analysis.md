# 八字事业分析指南

## 分析重点

做事业方向分析时，不建议只问“能不能成功”，而应重点看：
- 更适合稳定积累还是高波动路径
- 更适合独立推进还是依托平台
- 当前结构更重执行、管理、表达还是资源整合
- 大运流年是否容易引动职业变化

## 常见观察点

### 1. 日主状态
先看日主是否承得住事业压力与责任。

### 2. 官杀
看规则、压力、竞争、管理、职位感。

### 3. 财星
看资源承接、现实动机、经营意识。

### 4. 食伤
看表达、输出、创造、影响力。

### 5. 印星
看学习、吸收、支持系统、稳定性。

## 输出建议

可以按这个顺序讲：
1. 事业结构大致倾向
2. 更适合的职业类型
3. 容易遇到的阻力
4. 适合的发展方式
5. 阶段变化点

## 常见表达方式

- 更适合走长期积累型路径
- 更适合凭专业能力上升
- 更适合在规则明确的平台中发展
- 更适合先沉淀后放大
- 早期会有试错，中后期更看结构稳定度
