# 占卜时间点分析指南

## 这份文件做什么

用于帮助技能在处理“什么时候会变化”这类问题时，把时间判断说得更稳，不写成绝对预言，也不空泛拖过去。

## 分析重点

很多占卜问题，用户真正想知道的是：
- 什么时候会变化
- 转机什么时候出现
- 当前阶段还要拖多久
- 变化是突然发生，还是要先经过一个过渡期

## 常见判断方式

### 1. 近期
适合表示已经有变化信号，但还没完全落地。

### 2. 中期
适合表示需要经历一段过渡、等待或反复之后才会推进。

### 3. 阶段窗口
适合表示重点不在某一天，而在某个阶段、某轮沟通或某个条件成熟之后。

### 4. 延后判断
适合表示当前还没到真正转动的时候，继续强推效果也不大。

## 输出建议

时间点分析尽量讲：
- 近期
- 中期
- 某个阶段
- 某个窗口
- 哪种条件成熟后才更容易变化

尽量避免写成过度精确、像绝对预言一样的表达。

## 常见表达方式

- 近期会有变化信号
- 真正的推进更像发生在后一个阶段
- 当前还在拖延或反复期
- 转机点不在当下，而在下一轮沟通之后
- 不是没有变化，而是还没到真正落地的时候
